Components

This section contains a list of component devices used in this project. When possible I included a product link to where I purchased the device.

Pico

  • pico

LoRa Breakout Board

  • Adafruit RFM95W LoRa Radio Transceiver Breakout - 868 or 915 MHz - RadioFruit

Solar Charger

  • Adafruit Universal USB / DC / Solar Lithium Ion/Polymer charger - bq24074

  • Small 6V 1W Solar Panel

LoRa E5 Dev Board

  • Wio-E5 Dev Kit - STM32WLE5JC, ARM Cortex-M4 and SX126x embedded, supports LoRaWAN on EU868 & US915

Battery

  • Lithium Ion Cylindrical Battery - 3.7v 2200mAh

Photo cell

  • Photo cell (CdS photoresistor)

Transistors

  • NPN Bipolar Transistors (PN2222) - 10 pack
